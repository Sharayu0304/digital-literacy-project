## 4) MODULE SUMMARIES 
   MODULE 1 - HANDS ON TRAINING ON PRESENTATION SOFTWARE 
     This modules gives an introduction of digital literacy and how it is important in diffewrent professional environments. It disucsses various digital tools such as Canva, Prezi and Miro. This was applied to work on task 1 of this project. It was used to create an infographic on digital awareness. 

   + MODULE 2 - BUILDING A DIGITAL PORTFOLIO 
     This module discusses on the importance of maintaining a professional online presence using various platformsa such as LinkedIn, GitHub or Kaggle. It also explains how these platforns are used to discuss skills, making connections and improving career. 

  +  MODULE 3 - PROGRAMMING PLATFORMS AND COLLABORATIVE LEARNING PLATFORMS 
      This module explains how to Google Workspace - tools such as Google Forms, Docs, Sheets, and Slides. In task 3 we use Google forms to create a google quiz on digital awareness.The module helped in understanding how these tools support learning, collaboration, and data collection.

  +  MODULE 4 - EMAIL ETIQUETTE AND THE BASICS OF SOCIAL MEDIA ETIQUETTE 
      This module discussed the importance of email etiquette, guidelines of email etiquette, its structure and formats and how we should etiquesttes in a workspace. All these concepts were apllied into task 4 where 2 emails had to be written and also disscuss the DO's and DONT's on Social media platforms.

   +  MODULE 5 - AWARENESS ON CYBERCRIME & FINANCIAL FRAUDULENT 
       This module describes what cybercrims is, its types and financial fraudulents. It also explains the preventive measures and how to report if one encounters any such situation. All these concepts were apllied in task 5 to write a case study on UPI fraud payments and to make a prevention checklist. The module highlighted the importance of staying alert and protecting personal and financial information online.
